
# Library Implementation (C++)

## Overview
Minimal C++ library system with `LibraryItem`, `Series`, and `Volume` classes.

## Features
- Up to **3 volumes** in a series.
- **Price** & **world-building** updated on adding volumes.

## Usage
```cpp
Series s("Fantasy");
s.addVolumeToSeries("Book1", 19.99, 4.5);
s.display();
```
